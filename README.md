# 📊 Sistema de Gestão (Banco de Dados)

Este projeto contém a modelagem de um banco de dados relacional para gerenciamento de pessoas, funcionários, alunos, produtos, compras e pagamentos, incluindo especializações de produtos como eletrônicos, veículos e eletrodomésticos.

# 🔗 Exemplos Relacionamentos

* Uma Pessoa pode ser:

  - um Funcionário

- um Aluno

* Um Produto pode ser:

- Smartphone

- Computador

- Eletrodoméstico

- Veículo

* Um Pagamento está associado a:

- uma Pessoa

- um Produto
